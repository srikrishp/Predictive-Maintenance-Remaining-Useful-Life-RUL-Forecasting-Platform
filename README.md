# Predictive-Maintenance-Remaining-Useful-Life-RUL-Forecasting-Platform

## Project Overview
This project uses machine learning to predict engine failures before they happen using the NASA CMAPSS dataset.

The system:
- Predicts Remaining Useful Life (RUL)
- Detects anomalous engine behavior
- Recommends maintenance actions

## Technologies Used
- Python
- Pandas
- Scikit-learn
- XGBoost
- Isolation Forest
- Matplotlib
- Seaborn
- Google Colab

## Dataset
NASA CMAPSS Turbofan Engine Dataset

## Model Performance
- MAE: 29.69 cycles
- RMSE: 41.56 cycles
- Detected anomalies: 1032

## Workflow
Data Collection → Preprocessing → Feature Engineering → Model Training → RUL Prediction → Anomaly Detection → Maintenance Recommendation
